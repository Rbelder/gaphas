GTK+ View¶

This part describes the API of Gaphas.

mod:gaphor.view
class gaphas.view.GtkView(canvas=None, hadjustment=None, vadjustment=None)¶

GTK+ widget for rendering a canvas.Canvas to a screen. The view uses Tools from tool.py to handle events and Painters from painter.py to draw. Both are configurable.

The widget already contains adjustment objects (hadjustment, vadjustment) to be used for scrollbars.

This view registers itself on the canvas, so it will receive update events.
