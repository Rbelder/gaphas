Matrix¶

The Matrix class is used to define transformations on an item, relative to the parent Item. This is basically the same implementation as cairo.Matrix, only notifications are sent on state changes (see the state module).

mod:gaphor.matrix
class gaphas.matrix.Matrix(xx=1.0, yx=0.0, xy=0.0, yy=1.0, x0=0.0, y0=0.0)¶

Matrix wrapper. This version sends @observed messages on state changes

>>> cairo.Matrix()
cairo.Matrix(1, 0, 0, 1, 0, 0)
>>> Matrix()
Matrix(1, 0, 0, 1, 0, 0)
