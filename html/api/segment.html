class gaphas.segment.LineSegment(item, view)¶
merge_segment(segment, count=2)¶

Merge two (or more) item segments.

Tuple of two lists is returned, list of deleted handles and list of deleted ports.

Parameters:
segment

Segment number to start merging from (starting from zero).

count

Amount of segments to be merged (minimum 2).
split(pos)¶
split_segment(segment, count=2)¶

Split one item segment into count equal pieces.

Two lists are returned

  • list of created handles
  • list of created ports
Parameters:
segment

Segment number to split (starting from zero).

count

Amount of new segments to be created (minimum 2).
class gaphas.segment.SegmentHandleFinder(item, view)¶

Find a handle on a line, create a new one if the mouse is located between two handles. The position aligns with the points drawn by the SegmentPainter.

class gaphas.segment.SegmentHandleSelection(item, handle, view)¶

In addition to the default behaviour, merge segments if the handle is released.

class gaphas.segment.LineSegmentPainter(item, view)¶

This painter draws pseudo-hanldes on gaphas.item.Line objects. Each line can be split by dragging those points, which will result in a new handle.

ConnectHandleTool take care of performing the user interaction required for this feature.
