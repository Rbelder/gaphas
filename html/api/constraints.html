Constraints¶

The default constraint classes provided by Gaphas

mod:gaphor.constraint
class gaphas.constraint.Constraint(*variables)¶

Constraint base class.

  • _variables - list of all variables
  • _weakest - list of weakest variables
create_weakest_list()¶
Create list of weakest variables.
mark_dirty(v)¶
Mark variable v dirty and if possible move it to the end of Constraint._weakest list to maintain weakest variable invariants (see gaphas.solver module documentation).
solve_for(var)¶
Solve the constraint for a given variable. The variable itself is updated.
variables()¶
Return an iterator which iterates over the variables that are held by this constraint.
weakest()¶
Return the weakest variable. The weakest variable should be always as first element of Constraint._weakest list.
